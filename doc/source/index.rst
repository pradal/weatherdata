What is weatherdata ?
----------------------
WeatherData is a Python package that transforms weather data retrieved from agroservice in a native and
efficient Python data structure (Xarray.Dataset) to facilitate its usage in Python and OpenAlea. It will
provides a high-level interface to query, access, plot and compute several indicators in an efficient way. It
will also provides a uniform interface to the different weather data used in OpenAlea and elsewhere.
